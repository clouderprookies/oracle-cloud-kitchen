# Cloud Kitchen Playbook - PT-BR

## Princípios Fundamentais

- Foco em entrega de valor rápida via sprints
- "Configure First" sempre que possível
- Documentação mínima viável e viva no GitHub
- Colaboração intensa com o cliente
- Uso inteligente de IA como acelerador

## Ciclo de Entrega

1. Discovery Kitchen (1-2 sprints)
2. Foundation Station
3. Process Line
4. Extension & Automation
5. Data Migration Bay
6. Integration Hub
7. Testing & Adoption
8. Go-Live & Hypercare

## Regras de Ouro
- Todo To-Be deve ser colaborativo no repositório
- Decisões importantes registradas no Decision Log
