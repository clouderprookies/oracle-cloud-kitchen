# 📋 Aceleradores As-Is

Esta pasta contém **templates padrão** para mapear o estado atual (As-Is) dos processos de negócio.

**🎯 Objetivo:**
Ajudar a acelerar a fase de descoberta quando o cliente não tem processos documentados. Os templates trazem descrições simples, objetivas e baseadas em melhores práticas do Oracle Cloud ERP.

### Como usar
- Escolha o processo desejado
- Preencha as seções com as informações do cliente
- Use como base para workshops e entrevistas
- Depois avance para o To-Be

### Processos Disponíveis
- **Procure-to-Pay (P2P)** 🛒
- **Order-to-Cash (O2C)** 💰
- **Record-to-Report (R2R)** 📊

**💡 Dica:** Comece sempre pelo fluxo principal e depois detalhe os subprocessos.

## 🇺🇸 English Version

# 📋 As-Is Accelerators

This folder contains **standard templates** to map the current state (As-Is) of business processes.

**🎯 Objective:**
To speed up the discovery phase when the client has no documented processes. The templates provide simple, objective descriptions based on Oracle Cloud ERP best practices.

### How to Use
- Choose the desired process
- Fill in the sections with client information
- Use as a basis for workshops and interviews
- Then move on to the To-Be

### Available Processes
- **Procure-to-Pay (P2P)** 🛒
- **Order-to-Cash (O2C)** 💰
- **Record-to-Report (R2R)** 📊

**💡 Tip:** Always start with the main flow and then detail the subprocesses.
